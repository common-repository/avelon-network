== Description ==

= Advertise with Avelon =

Find new affiliates and grow your audiences with ease, save time & increase sales

- Zero fees. Just pay commission to affiliates with no onboarding or monthly costs
- Automated acceptance, reporting, tracking & monthly payment runs
- Track your existing ambassadors' ROI with 0% commission links

[Sign up for an Avelon Network Account](https://avelonetwork.com/signup/)

= How to use =

Simply log into your Avelon Network account. You can find the required settings in your Avelon Network Account under [Settings](https://app.avelonetwork.com/provider/settings/).

== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won't need to leave your web browser. To do an automatic install of Avelon Network, log in to your WordPress dashboard, navigate to the Plugins menu, and click "Add New".
 
In the search field type "Avelon" then click "Search Plugins." Once you've found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click "Install Now", and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the Avelon Network plugin and uploading it to your web server via an FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Updating =

Automatic updates should work smoothly, but we still recommend you back up your site.
